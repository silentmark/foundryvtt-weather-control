Calendar/Weather becomes Weather Control.

Calendar/Weather has been heavily reworked and has become Weather Control.

Weather Control is the same thing as Calendar/Weather V4, but under a different name and a new focus in mind.

You should already see a window with more information about Weather Control, but if you don you can read this Patreon post describing precisely what is going on.

In a nutshell, make sure you have Weather Control V4 installed, and you can uninstall Calendar/Weather. If you wish to go back to Calendar/Weather V3.1.7, please follow this guide.

If you need help, please head to The Bird's Discord server.
